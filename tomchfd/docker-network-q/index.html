docker network 问题

Q: 如图, 宿主机和bridge里的某一个容器是如何实现通信的

A: 在给 veth pair虚拟网卡设备配置好 ip 后,就会在路由表中添加 ip 和 mac 的映射关系,当 一个 ip 数据包从宿主机发给容器, 会通过 arp 协议 或查路由表, 得知要往哪个veth 网卡设备发出去包 (一一对应的) , 然后数据就会从 pair 的另一端收到 , 达到容器中

宿主机和容器的通信像网桥 , 容器和容器的通信像交换机
